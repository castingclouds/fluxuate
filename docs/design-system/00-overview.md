# Fluxuate Design System

This design system provides a consistent set of patterns and components for building user interfaces in Fluxuate. It includes support for both light and dark modes, accessibility considerations, and proper integration with Turbo Frames.

## Core Principles

1. **Consistency**: Use consistent spacing, typography, and color patterns
2. **Accessibility**: Ensure proper color contrast and interactive states
3. **Dark Mode**: Full support for both light and dark color schemes
4. **Responsiveness**: Layouts that work across different screen sizes
5. **Turbo Integration**: Proper integration with Hotwire/Turbo

## Usage

This design system is built on top of Tailwind CSS. All components use utility classes that can be copied and pasted directly into your views. The system is designed to work seamlessly with Ruby on Rails and Hotwire.
